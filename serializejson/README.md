Projeto Gson Java

Projeto simples em Java utilizando Maven e a biblioteca Gson para serializar objetos em JSON.

Dependência utilizada

Gson versão 2.13.2
https://mvnrepository.com/artifact/com.google.code.gson/gson/2.13.2

Estrutura

Classe Pessoa com:

String name
int idade

Funcionamento

No método main:

1. Criamos um objeto Pessoa
2. Utilizamos Gson para converter o objeto em JSON
3. Exibimos o JSON no console
